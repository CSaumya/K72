# 🚀 K72 UI Clone

A modern **UI clone of the K72 website**, built with a focus on smooth animations, clean layout, and responsive design.
This project recreates the UI of the K72 website using modern frontend technologies. It focuses on **animation, performance, and component-based architecture**.

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 Tailwind CSS
* 🎬 GSAP

## 🎯 Features

* Smooth animations using GSAP
* Clean and minimal UI
* Fully responsive design
* Interactive elements

This project is a **UI clone for learning purposes only**.
All design credits go to the original K72 website.
